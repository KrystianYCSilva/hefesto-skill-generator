# Kotlin Version History

Key features introduced in major Kotlin releases.

## Early Milestones
- **1.0 (2016)**: First stable release. Java interoperability, Null safety.
- **1.1 (2017)**: Coroutines (experimental), Type aliases.
- **1.2 (2017)**: Multiplatform projects (experimental).
- **1.3 (2018)**: Coroutines (stable), Inline classes (experimental), Unsigned integers.

## Modern Era
- **1.4 (2020)**: SAM conversions for Kotlin interfaces, Explicit API mode.
- **1.5 (2021)**: JVM records support, Sealed interfaces, Inline classes (stable).
- **1.6 (2021)**: Sealed when statements (exhaustive), Suspending functions as supertypes.
- **1.7 (2022)**: K2 compiler (alpha), Definitely non-nullable types (`T & Any`).
- **1.8 (2022)**: `data object`, New memory manager for Native (stable).
- **1.9 (2023)**: K2 compiler (beta), Enum class `entries` (replacement for `values()`).

## Current & Future
- **2.0 (2024)**: K2 compiler (default) - faster compilation, smarter analysis.
