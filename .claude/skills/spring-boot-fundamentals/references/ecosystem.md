# Spring Boot Ecosystem & Libraries

Comprehensive toolkit for modern Spring Boot development.

## Core Starters
- **spring-boot-starter**: Core starter, includes logging, auto-configuration, and YAML support
- **spring-boot-starter-web**: Starter for web applications including RESTful services
- **spring-boot-starter-data-jpa**: Starter for using Spring Data JPA with Hibernate
- **spring-boot-starter-security**: Starter for using Spring Security
- **spring-boot-starter-test**: Starter for testing with JUnit, Mockito, etc.
- **spring-boot-starter-actuator**: Starter for production-ready features

## Web & API Development
- **Spring Web MVC**: Traditional web framework
- **Spring WebFlux**: Reactive web framework
- **Spring Data REST**: Exposes Spring Data repositories as REST endpoints
- **Spring HATEOAS**: Support for building hypermedia-based RESTful web services

## Data Access
- **Spring Data JPA**: Simplifies data access layer implementation
- **Spring Data JDBC**: Lightweight alternative to JPA
- **Spring Data MongoDB**: MongoDB integration
- **Spring Data Redis**: Redis integration

## Security
- **Spring Security**: Authentication and authorization framework
- **Spring Security OAuth2**: OAuth2 support
- **Spring Session**: Session management

## Testing
- **Spring Boot Test**: Testing support with JUnit
- **Spring Test**: Spring testing framework
- **Mockito**: Mocking framework
- **Testcontainers**: Docker-based integration testing

## Monitoring & Operations
- **Spring Boot Actuator**: Production-ready features
- **Micrometer**: Metrics collection facade
- **Spring Cloud Sleuth**: Distributed tracing
- **Spring Cloud Netflix**: Integration with Netflix OSS components

## Cloud Platforms
- **Spring Cloud**: Tools for building distributed systems
- **Spring Cloud AWS**: Amazon Web Services integration
- **Spring Cloud GCP**: Google Cloud Platform integration
- **Spring Cloud Azure**: Microsoft Azure integration
